# How to Use

Used in the module:

```ts
import {LayoutModule} from 'ng-devui';
```

In Global Styles, introduce:
```css
@import '~ng-devui/devui-layout.css';
```

Use in the page:

```html
<!-- Grid -- >
<d-row>
  <d-col>
  </d-col>
</d-row>

<!-- Flex. The [dFlex] flag is added to the grid container by default. -- >
<xxx [dFlex]>
</xxx>

<!-- Space -->
<xxx [dSpace]="xxxx">
  <xx>
  </xx>
  <xx>
  </xx>
</xxx>

<!-- Gutter -->
<xxx [dGutter]="xxxx">
  <xx>
  </xx>
  <xx>
  </xx>
</xxx>

<!-- Class -->
<xxx [dClass]="xxxx">
</xxx>

<!-- Style -->
<xxx [dStyle]="xxxx">
</xxx>

<!-- If you need to use the layout container -- >
<d-layout>
  <d-header></d-header>
  <d-content></d-content>
  <d-footer></d-footer>
</d-layout>
```


# d-row
Grid layout row container. The `dFlex` directive is bound by default and is set as a row container. You can use the `dFlex (bound by default), dPace, dGutter, dClass, and dStyle` directives for more layout settings.

# d-col
Grid layout column container, expected to be a child element of `d-row`. The `dFlex` directive is bound by default. More layout settings can be made using the `dFlex (bound by default), dPace, dGutter, dClass, dStyle` directives.

# d-layout

Layout container, which can be combined with `d-header`, `d-content`, `d-footer`, `d-aside` to implement layout.
Elements that can be nested under `d-layout`: `d-header`, `d-content`, `d-aside`, `d-layout`.

# d-header

Top layout, which can be implemented only in the `d-layout` container as the top of the `d-layout` container.
Default height: 40 px

# d-footer

Bottom layout, which can be implemented only in the `d-layout` container as the bottom of the `d-layout` container.

# d-content

Content container, which can be placed only in the `d-layout` container as the content between `d-header` and `d-footer` in the `d-layout` container.

# d-aside

Sidebar, which can only be placed in the `d-layout` container as the sidebar part of the `d-layout` container.
