基本用法
通过 options 配置数据源
通过 overview、size 配置样式
通过 direction 配置展开方向
使用对象
使用对象数组，通过 filterKey 配置对象显示字段
自定义搜索功能
通过 isSearch、searchFn开启搜索功能并自定搜索方法
通过 multiple 启用多选功能
全选下拉选项
多选下拉中通过 isSelectAll 启用全选功能
自定义模板
通过模板配置下拉选项和已选项
标签化
通过 extraConfig.labelization 配置标签化：
1.enable为true时启用，已选项显示为可删除的标签
2.overflow配置超长的样式，具体参考api
禁用
通过 disabled 禁用下拉，
通过选项对象的$disabled禁用选项，
通过选项对象的$immutable禁止修改选项
允许清空值
附着到Body上
使用 appendToBody 使得附着到body上，不被带滚动条的容器遮挡
虚拟滚动 或 懒加载
使用 virtualScroll 在大数据量下用虚拟滚动；
使用 enableLazyLoad 在大数据量下用懒加载
异步加载显示加载中
自定义区域
使用 customViewTemplate 定义自定义区域
自定义区域方向和选中
设置已选项顺序源数组顺序或选中顺序
通过 keepMultipleOrder 配置多选的时候已选项维持原数组顺序：'origin'源数组排序,'user-select'用户选择顺序排序
用户场景：限制选中个数
用户场景：自定义搜索的懒加载
用户场景：自定义搜索和模板结合 —— 邮箱搜索
