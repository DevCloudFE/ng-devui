基本用法
简单表格，展示列表数据
表格交互
支持列排序、列过滤、选中、调整列宽
异步加载数据
异步加载表格数据，列动态配置
表头固定
设置表格最大高度，同时表头不跟随滚动
列表数据懒加载
跟随滚动，懒加载，不断增加表格数据
表头分组
表头支持分组，显示多行
编辑单元格
通过列的编辑属性（editable）和自定义单元格来实现单元格编辑功能
扩展额外行
通过headerExpandConfig和行数据的$expandConfig来配置表头和某一行下的可自定义的扩展行，通过showExpandToggle来配置扩展行的操作图标， 也可以通过设置行数据的xpandConfig.expand来控制扩展行的显示和隐藏
树形表格
通过列的nestedColumn属性和行数据的children字段来配置渲染树形表格
固定列
当表格列过多时，固定列有利于用户在左右滑动屏幕时，能够便捷地进行数据定位与对比，通过column的fixedLeft、fixedRight来配置
列拖拽
应用于列内容有对比诉求，属于低频操作，默认不展示，当鼠标移入表头显示标识，支持用户快速对列进行排序
