基本用法
简单表格，展示列表数据
支持两种实现方式，方式一通过自定义head、body模板和其中的行、单元格来实现，方式二通过配置column来实现
表格样式
异步加载数据、配置列及空数据模板
模拟异步加载数据、支持配置动态列、使用#noResultTemplateRef配置空数据模板
表格交互
支持列排序、列过滤、自定义过滤模板、选中、调整列宽
支持两种实现方式，方式一通过自定义head、body模板和其中的行、单元格实现，方式二通过配置column来实现
自定义表格选中操作
通过设置checkOptions来配置表头选中的下拉项及操作
支持两种实现方式，方式一通过自定义head的行来实现，方式二通过配置column来实现
懒加载
使用lazy启用懒加载，当滚动表格底部时到分发loadMore事件实现懒加载
虚拟滚动
使用virtualScroll启用虚拟滚动，行高须确定且相同，如果行高高于默认的40px，须配置virtualItemSize
若初始状态无数据，需要对class为`cdk-virtual-scroll-viewport`的元素设定高度，并且建议对`d-data-table`元素和noResult模板设定定位
表头固定
使用fixHeader设置表头固定，使之不跟随滚动
表头固定虚拟滚动
使用fixHeader设置表头固定，并启动虚拟滚动，高度自适应
表头分组
支持两种实现方式，方式一通过自定义head模板中th的colspan、rowspan来实现，方式二通过配置column的advancedHeader来实现
编辑单元格
支持两种实现方式，方式一通过自定义body模板中的单元格来实现，方式二通过配置column来实现
扩展行
支持两种实现方式，方式一通过自定义body模板中的行来实现，方式二通过配置column来定义table，并配置input和行数据来实现
树形表格
通过nestedColumn属性和行数据的children、$isChildTableOpen字段来配置渲染树形表格
支持两种实现方式，方式一通过自定义head、body模板和其中的行、单元格来实现，方式二通过配置column来实现
固定列
当表格列过多时，固定列有利于用户在左右滑动屏幕时，能够便捷地进行数据定位与对比，通过fixedLeft、fixedRight来配置
支持两种实现方式，方式一通过自定义head、body模板和其中的行、单元格来实现，方式二通过配置column来实现
列拖拽
应用于列内容有对比诉求，属于低频操作，默认不展示，当鼠标移入表头显示标识，支持用户快速对列进行排序
单元格合并
支持使用colspan、rowspan等属性来设置单元格的合并，只能通过自定义head、body模板和其中的行、单元格来实现
行拖拽
支持使用DragDrop组件来实现表格的行拖拽功能，只能通过自定义head、body模板和其中的行、单元格来实现
批量行拖拽
支持使用DragDrop组件来实现表格的批量行拖拽功能，请用Ctrl按键和鼠标多选并进行拖拽，只能通过自定义head、body模板和其中的行、单元格来实现
