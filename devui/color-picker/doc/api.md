### d-color-picker 参数
|  参数   |   类型    | 默认 | 说明                             |
| :-----: | :-------: | :--: | :------------------------------- |
|  hide   | `boolean` | true | 【可选】控制显隐                 |
|  color  | `string`  |  ""  | 【可选】初始颜色                 |
| cancel  |  `func`   |  --  | 【可选】取消选色并关闭选色器     |
|  send   |  `func`   |  --  | 【可选】获取取色器输出色值       |
| confirm |  `func`   |  --  | 【可选】确认取色色值并关闭选色器 |
