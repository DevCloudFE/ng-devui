### d-input-number 参数

| 参数        | 类型          | 默认        |   说明                 |
| :---------: | :----------: | :---------: | :----------------------------------------------------------------------|
| max  | `number`  | 100     | 最大值 |
| min  | `number`  | 0     | 最小值 |
| step  | `number`  | 1     | 步进值 |
| disabled  | `boolean`  | false     | 禁止输入态开关 |
| size  | `string` | --     | 组件大小 `['sm', 'lg','']`  |
| ngModel  | `number` | --     | 组件的值 |
